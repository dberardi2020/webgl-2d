# webgl-2d

An interactive 2D line drawing tool built with raw WebGL. College graphics course project.

**[Live Demo](https://dberardi2020.github.io/webgl-2d/)**

## Features
- Draw mode — click to place vertices and draw connected lines
- File mode — load `.dat` files to render pre-defined shapes
- Color picker and keyboard shortcuts

## Controls
| Key | Action |
|-----|--------|
| `F` | Switch to file mode |
| `D` | Switch to draw mode / clear canvas |
| `C` | Cycle drawing color |
| `B + Click` | Start a new line |
